# Naming Convention

Terse style guide for agent asset naming conventions.

## Skills

1. **Standard Skills**:
   - Format: `<domain>-<verb>-<details>.md`
   - Example: `create-agent-skill.md`

2. **Routing Skills** (containing sub-skills):
   - Format: `<domain>-<details>-<location>.md`
   - Locations: `console`, `studio`
   - Example: `agent-asset-studio.md`
