# Emojis Reference

Use emojis to optimize token consumption for models and improve visual scannability for humans.

## Emojis for LLMs

Semantic emojis that replace natural language to reduce token usage:

### Status

- `✅` Done / Pass / Success
- `❌` Fail / Error
- `⚠️` Warning
- `ℹ️` Info / Note
- `⏳` Pending / In Progress

### Actions

- `🔍` Inspect / Search
- `📝` Write / Edit / Create
- `🗑️` Delete / Remove
- `🔄` Sync / Update / Loop
- `🧪` Test / Verify
- `🚀` Deploy / Release / Push

## Emojis for Humans

Color and shape emojis to improve scannability for human readers:

### Priority & Status Indicators

- `🔴` High Priority / Critical / Stop / Delete
- `🟡` Medium Priority / Warning / Pending
- `🟢` Low Priority / Safe / Go / Done
- `🔵` Info / Reference / Guide

### Shapes & Colors

- **Circles**: `🔴` `🟠` `🟡` `🟢` `🔵` `🟣` `⚫` `⚪` `🟤`
- **Squares**: `🟥` `🟧` `🟨` `🟩` `🟦` `🟪` `⬛` `⬜` `🟫`
- **Hearts**: `❤️` `🧡` `💛` `💚` `💙` `💜` `🖤` `🤍` `🤎`
