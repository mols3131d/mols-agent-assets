# Google Advanced Search

## Operators

- `"query"`: Exact match.
- `-term`: Exclude term.
- `site:domain`: Limit to domain/TLD.
- `filetype:ext`: Filter extension.
- `intitle:term`: Title match.
- `inurl:term`: URL match.
- `after:YYYY` / `before:YYYY`: Filter by year.
- `A AROUND(N) B`: Proximity match.
- `*`: Wildcard match.
- `..`: Number range filter.
- `OR`: Logical OR.

## Multi-word Values

- Wrap values containing spaces in double quotes `""` (e.g., `-"exclude phrase"`, `intitle:"phrase match"`).
