# Agent Workflows

**에이전트의 명시적 작업 절차(Pipeline) 및 파이프라인 저장소.**

## 1. Description

- **성격**: 액티브(Active) 및 순차적 실행기. 명확한 **Step-by-Step** 프로세스로 결과 도출.
- **예시**: `/git-commit`, 배포 파이프라인 등 "무엇(What)을 어떤 순서로 할 것인가"에 대한 절차.

## 2. vs Skills

- **Workflows**: 명확한 시작과 종료의 순차적(Sequential) 1, 2, 3 단계 요구.
- **Skills**: 패시브 형태의 방법론 적용 (지속적 상태/원칙 유지).
