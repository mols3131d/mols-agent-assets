# Overhand Rules

`.agents/brain/overhand.md` (세션 인수인계) 운영 규칙.

## 1. Principles

- **고밀도 압축**: 다음 에이전트가 즉각적으로 맥락을 파악할 수 있도록 토큰 최소화.
- **단발성 유지**: 세션이 전환될 때마다 이전 내용은 폐기하고 최신 상태로 덮어쓰기 (Context Tax 방지).

## 2. Format

1. **Goal**: 현재 도달하고자 하는 최종 목표.
2. **Done**: 방금 종료된 세션에서 완료된 핵심 사항.
3. **To-Do**: 새 세션 시작 즉시 실행해야 할 구체적 Action Item.
4. **Context/Issue**: 직전까지 발생한 에러, 시도했던 해결책 (KISS/DRY 기반).

## 3. Trigger

- 사용자가 작업 중단/휴식을 선언하거나, 에이전트가 세션 종료를 판단할 때 자동 작성.
- 새 세션 시작 시 가장 먼저 읽고 작업 재개.
