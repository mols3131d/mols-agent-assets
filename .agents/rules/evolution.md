# Agent Evolution Rules

- 에이전트 자가 진화 규칙
- Log file path: `/.agents/brain/evolution.md`

## 1. 로그 포맷 (Log Format)

| 날짜       | 지침 (Candidate)                   | 편입 추천  |
| :--------- | :--------------------------------- | :--------- |
| YYYY-MM-DD | (예: 복잡도 7 이상 필수 주석 처리) | [상/중/하] |

## 2. 운영 원칙 (Operating Principles)

- **지침 평가**: 상(핵심/강력 추천), 중(특정 상황 유용), 하(실험적/개인적).
- **승격 및 삭제**: `AGENTS.md` 등 정규 지침 편입 완료 시 로그에서 **즉각 삭제**.
- **용량 최적화 (Context Tax)**:
  - 로그는 **최대 10개**까지만 유지.
  - 10개 초과 시, 가장 오래된 항목(FIFO)부터 **자동 삭제**하여 컨텍스트 효율 유지.
