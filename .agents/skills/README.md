# Agent Skills

**에이전트의 자율적 능력(Capability) 및 방법론(How-to) 저장소.**

## 1. Description

- **성격**: 패시브(Passive) 및 자율 발동기. 특정 상황(Trigger)에서 에이전트가 스스로 판단하여 적용.
- **예시**: 문서 밀도 압축(DCO), 토큰 최적화 등 "어떻게(How) 작업할 것인가"에 대한 기술.

## 2. vs Workflows

- **Skills**: 상태/방식 유지 (지속적 원칙). 순서 없음.
- **Workflows**: 순차적 파이프라인 (명시적 절차). Step 1 -> 2 -> 3.
