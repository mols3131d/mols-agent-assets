# 🔄 워크플로우 명칭

이 문서는 [특정 결과물]을 생성하거나 작업을 완수하기 위한 표준 절차를 정의합니다.

## 📌 핵심 목적

이 워크플로우를 통해 달성하고자 하는 비즈니스 가치와 실행 효율성을 설명합니다.

## 📜 실행 단계 (WLM)

### 1. 탐색 [SCAN]
- 작업 착수 전 환경과 현재 상태를 파악하는 방법을 설명합니다.

### 2. 계획 [PLAN]
- 작업을 어떤 전략으로 수행할지, 어떤 규칙을 참조해야 하는지 정의합니다.

### 3. 실행 [EXEC]
- 실질적인 작업 수행 시 주의해야 할 점을 기술합니다.

### 4. 검증 [VERIFY]
- 작업 결과물이 성공적인지 판단하는 기준을 제시합니다.
