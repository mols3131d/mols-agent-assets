# 📋 표준 템플릿 (Templates)

이곳은 프로젝트 전반에서 사용되는 **일관된 문서 형식을 보장하기 위한 템플릿**들이 모이는 공간입니다.

## 📂 제공되는 템플릿

- **`/templates/doc.md`**: 일반 지식 문서 및 표준 정의용 설계도
- **`/templates/rule.md`**: 에이전트 규칙(RLM) 정의용 설계도
- **`/templates/workflow.md`**: 에이전트 워크플로우(WLM) 정의용 설계도

## 🚀 활용 가이드

- **일관성 유지**: 모든 문서가 동일한 구조(프론트매터 등)를 갖도록 강제하여 에이전트와 인간의 이해도를 높입니다.
- **작업 속도 향상**: 새로운 규칙이나 문서를 만들 때 기초 구조를 즉시 제공합니다.
- **이중화 준수**: 새로운 자산을 생성할 때 해당 템플릿을 복사하여 사용하고, 반드시 영문(`.md`)과 한국어(`.ko.md`) 쌍을 함께 생성하세요.

*이곳의 템플릿은 에이전트와 인간의 협업 효율을 극대화하기 위한 설계도입니다.*
