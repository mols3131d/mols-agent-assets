# Studio Workspace

이 폴더는 자유로운 실험과 작업이 이루어지는 **에이전트 전용 작업 공간**입니다.

## 에이전트 지시사항 (Agent Instructions)

- **규칙 예외**: 이 폴더 내에서 이루어지는 모든 작업 및 아티팩트(`implementation_plan.md`, `task.md` 등) 작성 시, 루트의 공통 규칙(예: 한국어 강제 사항)보다 **본 가이드라인이 우선**합니다.
- **자율성**: 이 폴더 내에서는 에이전트가 가장 효율적이라고 판단하는 언어와 도구를 자유롭게 사용할 수 있습니다.
- **용도**: 프로토타이핑, 코드 생성 실험, 임시 데이터 분석 등.
