# mols-agent-assets

AI 에이전트 자산(스킬, 프롬프트, 규칙, 에이전트)을 개발, 검증 및 관리하는 저장소입니다.

---

## 자산 유형 (Asset Types)

| 자산 유형 | 설명 |
| :--- | :--- |
| `agents` | 에이전트 역할, 목표 및 사용 가능 도구를 정의하는 구성 |
| `skills` | 특정 작업을 반복 수행하기 위한 절차적 워크플로, 스크립트, 가이드 |
| `rules` | 에이전트 동작 및 코드 스타일을 제어하는 공통 규칙 및 제약 조건 |
| `prompts` | 특정 작업을 위한 재사용 가능한 프롬프트 템플릿 |

## 저장소 구조 (Repository Structure)

| 디렉터리 | 설명 |
| :--- | :--- |
| `.agents/` | 에이전트 도구가 참조하는 런타임 지침 및 설정을 관리 |
| `src/` | 에이전트 자산을 신규 생성하고 테스트, 배포하는 기본 개발 공간 |
| `tests/` | 자산 및 스크립트를 검증하는 자동화 테스트 모듈 |
| `docs/` | 사람용 설명 문서 및 상세 가이드 |
| `scripts/` | 저장소 자동화 도구 스크립트 |
