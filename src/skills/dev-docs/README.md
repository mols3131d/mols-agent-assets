# 개발 문서 관리 스킬 (dev-docs)

이 스킬은 프로젝트의 의사결정 기록(ADR) 및 기능 명세(SPEC) 작성을 지원합니다.

## 주요 구성 요소

- **ADR (Architecture Decision Record)**: 아키텍처 결정 사항 및 트레이드오프 기록. (`docs/adr/`)
- **SPEC (Specification)**: 상세 설계 및 구현 가이드. (`docs/spec/`)

## 사용 방법

에이전트는 새로운 기능을 개발하거나 기술적 결정을 내릴 때 이 스킬의 지침을 따라 문서를 작성합니다. 상세한 규칙은 [SKILL.md](./SKILL.md)를 참조하십시오.
