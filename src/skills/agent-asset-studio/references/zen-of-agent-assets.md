# The Zen of Agent Assets

Agent assets are not more instructions. They are interfaces for precise action with less context.

1. Concise beats verbose.
2. Deterministic beats nondeterministic.
3. Explicit beats implicit.
4. One clear job beats many duties.
5. Exact paths beat directory scans.
6. Executable scripts beat repeated prose.
7. One source of truth beats duplicate rules.
8. On-demand loading beats loading everything.
9. User intent beats keyword matching.
10. Boring defaults beat clever exceptions.
11. Verifiable results beat unsupported confidence.
12. Small behavior-preserving changes beat rewrites.
13. Reversible changes beat destructive ones.
14. Observable evidence beats guesses.
15. Needed assets beat empty future scaffolds.
16. Clear stop conditions beat silent errors.
17. Minimum sufficient workflows beat every possible combination.
18. Shared conventions beat novel ones.

Concise keeps meaning. Deterministic allows judgment. Simple never sacrifices safety, accuracy, or accessibility.

When rules conflict: user intent, safety, verifiability win. Make exceptions explicit. Record reason and boundary.
