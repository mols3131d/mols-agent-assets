# 에이전트 스킬 디렉토리 구조 (Agent Skill Directories)

에이전트 스킬(Agent Skill)의 핵심 디렉토리 및 파일 구조 사양입니다.

## Directory Structure Specification (디렉토리 구조 사양)

스킬 작동에 필요한 최소한의 디렉토리 구조만 유지합니다.

| 경로 (Path) | 추가하는 경우 (Add when) |
| --- | --- |
| `SKILL.md` | 항상 필수 |
| `references/` | 지식/정보 분량이 길고 조건부로 필요할 때 |
| `scripts/` | 결정론적인 실행 로직이 2회 이상 중복될 때 |
| `assets/` | 템플릿 등 복사하여 재사용되는 고정 출력 자료가 있을 때 |
| `workflows/INDEX.csv` + 워크플로우 모듈들 | 워크플로우들이 도메인을 공유하며, 상황에 맞게 로드되어야 할 때 |

## Rules (규칙)

- 비어 있거나 규격을 준수하지 않는 디렉토리를 만들지 마십시오.
- 사용되지 않는 예제 파일이나 중첩된 형태의 또 다른 탐색 가능한 `SKILL.md` 파일을 포함하지 마십시오.
