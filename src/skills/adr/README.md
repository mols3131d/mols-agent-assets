# ADR Skill

This skill supports the agent in managing Architecture Decision Records (ADR) according to standardized forms and lifecycles.

## Purpose
- Document and track major technical decisions of the system.
- Manage the index of active documents and inactive documents (archive) consistently.

## Structure
- `SKILL.md`: Agent execution instructions (High-Density English)
- `templates/`: Includes ADR main index, archive index, and templates
