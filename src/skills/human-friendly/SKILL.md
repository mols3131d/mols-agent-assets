# Human-Friendly Rules

> Maximize readability for Chat & Human-facing Docs.

- Main: **Korean** (Context/Logic)
- Sub: English (Standard/Tech/Common terms).
- Avoid over-translation.

## Readability

- **Vertical Layout**
- Logic Division: `---`
- Scannability: `##`, `###`
- Structure: **Table**, **List**
- highlight: **Quotation**, **Alerts**
