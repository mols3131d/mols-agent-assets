# SPEC Skill

This skill supports the agent in managing detailed functional specifications (SPEC) and implementation designs according to standardized forms and lifecycles.

## Purpose
- Design and record detailed requirements, data structures, APIs, etc., before functional implementation.
- Manage the index of active documents and inactive documents (archive) consistently.

## Structure
- `SKILL.md`: Agent execution instructions (High-Density English)
- `templates/`: Includes SPEC main index, archive index, and templates
