# PRD Skill

에이전트가 제품 요구사항 정의서(PRD)를 작성 및 관리하여, 기능 구현 전에 제품의 비전, 비즈니스 목적, 사용자 시나리오 및 기능적 요구사항을 표준화된 양식으로 기획할 수 있도록 지원하는 스킬입니다.

## 목적
- 사용자 및 비즈니스 관점에서 "무엇을, 왜 만들어야 하는가"를 명확하게 정의합니다.
- 기획자, 디자이너, 개발자, QA 간의 요구사항 정합성을 맞추고 일관된 목표를 공유합니다.
- 활성 및 보관용 PRD 문서의 생명주기를 시스템적으로 관리합니다.

## 구조
- `SKILL.md`: 에이전트 실행 지침 (High-Density English)
- `templates/`: PRD 메인 인덱스, 보관함 인덱스, 템플릿 포함
