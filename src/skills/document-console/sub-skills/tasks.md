# Tasks

Goal: turn work into clear actions.

## Use When

- User asks for todo/checklist.
- Existing doc needs action items.
- Work does not need kanban stages.

## Task Format

```md
- [ ] Action verb + object. Owner/date if known.
```

Rules:

- Start with verb.
- One task = one action.
- Keep context in linked doc, not task text.
- Group by milestone only when list exceeds ~10 items.
