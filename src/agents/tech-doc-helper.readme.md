# Tech Doc Helper Agent

[tech-doc-helper.md](tech-doc-helper.md)는 기술 문서 분석 및 한국어 번역을 위한 에이전트 자산입니다.

## 주요 기능

* **스캔 최적화**: 기술 문서를 요약 및 구조화하여 빠른 파악을 지원합니다.
* **기술 전문 번역**: 자연스러운 한글 번역을 제공하되, 표준 IT 용어는 영문 원문을 유지하거나 병기합니다.
* **맥락/스펙 보존**: 핵심 스펙, 수치, 아키텍처 흐름 및 마크다운 레이아웃을 누락 없이 보존합니다.
* **코드 주석 번역**: 소스 코드는 보존하고 내부 주석만 한글로 번역합니다.
* **Caveman-Lite 적용**: 미사여구와 불필요한 수식어를 제거하여 토큰 효율을 극대화합니다.

## 사용 방법

1. [tech-doc-helper.md](tech-doc-helper.md) 내용을 복사합니다.
2. AI 채팅봇의 시스템 프롬프트(System Prompt) 또는 첫 메시지에 주입합니다.
3. 분석할 문서(텍스트/URL)를 입력합니다.
