# 개발 워크스페이스 (`src/`)

AI 에이전트 자산(스킬, 프롬프트, 규칙, 에이전트)을 신규 생성, 수정, 검증 및 배포하는 통합 공간입니다.

---

## 지침

- **브랜치 활용**: `feat/<asset-name>` 형식의 피처 브랜치에서 에셋을 개발합니다.
- **구조**: 작업 단위 및 자산 유형별 하위 디렉터리를 생성하여 작성합니다.
- **검증**: `agent-asset-studio` 스킬로 에셋을 검증하고 `tests/` 디렉터리에 테스트 코드를 작성합니다.
- **배포**: 검증 완료 후 피처 브랜치를 `main` 브랜치로 커밋 및 머지하여 배포합니다.
