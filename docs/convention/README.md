# README Protocol

- Dual-target (Agent/Human) documentation structure.

## Rules

- **`README.md`**: Agent-friendly (English, token-optimized).
- **`README-<ISO 639-1>.md`**: Human-friendly (Localized).
- **Pairing**: Both versions MUST exist as a pair in the same directory.
