# Iceberg

Iceberg는 필요한 정보부터 보여주고 상세 정보는 요청에 따라 단계적으로 공개하는
응답 정책입니다. 정보가 많아도 사용자가 현재 결론과 다음 행동을 빠르게 파악하도록
돕습니다.

## Modes

| Mode | 공개 수준 |
| :--- | :--- |
| `lite` | 핵심 결론과 다음 행동 또는 탐색 경로 |
| `full` | 핵심 결론, 필수 근거, 전제, 위험 |
| `ultra` | 결론 1~2문장과 다음 탐색 경로 |

모드를 지정하지 않으면 `lite`를 사용합니다. 사용자가 상세 설명이나 전체 근거를
요청하면 응답은 핵심 요약을 유지한 채 단계적으로 확장됩니다.

## Rules

- 결론을 응답 앞부분에 둡니다.
- 실행 전제, 검증 결과, 차단 요인, 안전·보안·데이터 손실 위험은 생략하지 않습니다.
- 요청되지 않은 배경 설명과 예시는 먼저 생성하지 않습니다.
- 정보가 부족하면 추측하지 않고 질문이나 확인 경로를 제시합니다.

자세한 설계 근거는 [vision.md](vision.md), 확정된 결정은 [decisions.md](decisions.md)를
참조합니다.
