# WWH (Why-What-How)

## Purpose

Logical framework for driving decisions or actions.

## Building Blocks

1. **Why**: Intent & Value. Reason for action.
2. **What**: Outcome & Scope. The actual proposal.
3. **How**: Realization Logic. Executable path.

## Flexibility

- **Modular**: Skip blocks based on role (Planner -> skip How | Developer -> skip Why).
- **Non-Linear**: Any order (What -> Why, or Why -> What).
