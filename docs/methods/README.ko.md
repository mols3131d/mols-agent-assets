# 방법론 저장소 (Universal Methodology)

이 디렉토리는 특정 프로젝트의 운영 규칙이 아닌, **보편적이고 다른 환경에서도 즉시 참조 및 적용할 수 있는 범용 방법론**들을 보관합니다.

## 핵심 가치

1.  **보편성 (Universality)**: 특정 도구나 워크스페이스에 종속되지 않는 논리적 프레임워크를 지향합니다.
2.  **이식성 (Portability)**: 다른 프로젝트나 에이전트 환경에서 표준 지침으로 즉시 가져다 쓸 수 있는 고밀도 로직을 제공합니다.
3.  **참조성 (Referability)**: 복잡한 의사결정이나 설계 시 신뢰할 수 있는 '이론적 근거'가 됩니다.

## 포함된 방법론 (Methodologies)

- **[WWH](/docs/methods/WWH.ko.md)**: Why-What-How 기반의 논리 정렬 프레임워크.
- **[TAS](/docs/methods/TAS.ko.md)**: 정-반-합 구조의 사고 모델.
- **[TAS-Debate](/docs/methods/TAS-Debate.ko.md)**: 역할 분담을 통한 구조적 토론 프레임워크.
- **[PDD](/docs/methods/PDD.ko.md)**: 페르소나 기반의 의사결정 로직.
- **[PRD-SRS](/docs/methods/PRD-SRS.ko.md)**: 요구사항 정의 및 명세화 표준.
- **[RFC-ADR](/docs/methods/RFC-ADR.ko.md)**: 제안과 결정의 수명 주기 관리 모델.
