# SYNOPSIS

ACE ::= {Rule_Engineering | Workflow_Optimization | Evaluation_Logic}

guideline ::= {Management_Standards | Lifecycle_Control | Quality_Benchmarks} for ACE_Assets

# PROTOCOL

- **Context_as_Product**: Rules and workflows are engineered products, not simple text files.
- **Continuous_Refinement**: The goal is to maximize agent intelligence via high-density context engineering.
- **Systemic_Improvement**: Focus on the *process* of managing assets: Create -> Evaluate -> Improve -> Archive.

# TOPOLOGY

- rule/ :: Methodologies for engineering and managing the lifecycle of behavioral rules.
- workflow/ :: Standards for designing and optimizing executable procedural assets.

# CONSTRAINT

- Core_Objective: Better Agent Output via Better Context Management.
- Governance: Adhere to lifecycle standards defined in sub-directories.
