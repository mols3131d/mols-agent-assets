# ACE (Agent Context Engineering)

## Definition

Process of designing agent intelligence (cognition & behavior) as structured, manageable **algorithmic assets** (Rules, Workflows, Protocols) rather than flat prompts.

## Core Logic

- **Context = Intelligence**: The quality of structured information defines the output quality.
- **Engineering Over Prompting**: Focus on lifecycle management (Creation -> Improvement -> Evaluation).

## Key Pillars

1. **Modules**: Atomic logic blocks (Rules).
2. **Operations**: Step-by-step logic (Workflows).
3. **Protocols**: Communication and task-handling standards.

## Objective

High-fidelity, repeatable, and evaluable agent performance.
